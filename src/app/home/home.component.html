WR-Issue 

WR-Issue ist ein, von Webservice Rathje entwickeltes System, mit dem man Schäden in einem speziellen Bereich dem Zuständigen melden kann.  

Info

Warum ist unser System so gut?

Kompatibilität

Wir versuchen stetig unsere Handyapp für alle Geräte verfügbar zu machen. Außerdem wird das Administrationpanel dauerhaft für die neuen Features der bekanntesten Browser angepasst.

Vorteile

Hier finden sie die größten Vorteile unseres Systems.

Performance

Unser Backend ist besonders schnell und kann somit viele Daten auf einmal verarbeiten. Dies bietet unglaublich große Möglichkeiten für unsere Kunden.

Open Source

Da unser Frontend Open Source ist, kann die Sicherheit unseres Systems gewährleistet und stetig verbessert werden.

Kosten

Da war bei der Entwicklung unseres Systems auf Open Source setzten, können wir die Kosten niedrig halten und das System somit auch günstig anbieten.

Regelmäßige Updates

Wir updaten unsere Systeme ohne, dass dabei die Kompatibilität verloren geht. Außerdem wird dadurch die Sicherheit gewährleistet.
